# AGENTS.md

## Project Purpose

This project builds a **static HTML website using Astro**.

The site is deployed using **GitHub Pages**.

The build output must be a **fully static site** with no server-side runtime.

---

## Technology Stack

- Astro (Static Site Generator)
- HTML
- CSS
- JavaScript

No server-side framework is used.

---

## Deployment Target

The site must be compatible with **GitHub Pages**.

Therefore:

- The site must be generated as **static HTML**
- No server-side rendering
- No backend APIs
- All functionality must work in a static hosting environment

---

## Astro Requirements

Use Astro in **static output mode**.

Build command:


astro build


Output directory:


dist/


This directory contains the final static HTML site.

---

## Development Guidelines

Follow these principles:

- Prefer static rendering
- Use simple HTML structure
- Keep JavaScript lightweight
- Avoid unnecessary dependencies
- Ensure compatibility with static hosting

---

## Goal

Create a clean static website using Astro that can be deployed directly to **GitHub Pages**.
